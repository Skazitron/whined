## Whine - A white noise generation engine

> **This is a WIP**

A gRPC based noise generation engine built using Go supporting white noise, brown noise,
and pink noise. You are able to use up to 8 different noises with different cut-off
frequencies and volumes.

## Installation
- Use the install script
- Use the binary: the binaries are available to install for `x86` and `arm` based Linux, Windows, and MacOS.
